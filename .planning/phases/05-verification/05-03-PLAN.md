**Imports:**
- import asyncio, import json, import sys
- import typer
- from rich.console import Console
- from ascend_agent.config import settings
- from ascend_agent.diagnosis.models import ReproductionResult, VerificationResult
- from ascend_agent.diagnosis.router import ModelRouter
- from ascend_agent.verification.engine import VerificationEngine
- console = Console()
- verify_app = typer.Typer(name="verify", help="Verify fixes by running relevant tests")

**Command: verify_run (decorated with @verify_app.command(name="run")):**

Arguments:
- reproduction: str = typer.Argument(..., help="Path to reproduction result JSON")
- output: str | None = typer.Option(None, "--output", "-o", help="Path to write verification result as JSON")

Implementation (mirroring reproduce_run pattern):

1. **Load JSON:** try: with open(reproduction) as f: data = f.read(). except (FileNotFoundError, json.JSONDecodeError) as e → console.print error, raise typer.Exit(code=1)
2. **Parse:** reproduction_result = ReproductionResult.model_validate_json(data). except Exception → console.print error, raise typer.Exit(code=1)
3. **Get repo_path:** the verify CLI receives a ReproductionResult JSON file path (not a DiagnosisOutput). The repo_path should be determined from the Settings: repo_path = settings.repo_path or "." — unless reproduction_result has a repo path embedded. Use settings.repo_path or fallback to Path.cwd().
4. **Init engine:** try: router = ModelRouter(); engine = VerificationEngine(router=router, repo_path=repo_path, settings=settings). except ValueError as e → console.print error + hint for OPENAI_API_KEY, raise typer.Exit(code=1)
5. **Run:** console.print("\n[bold cyan]Running verification...[/bold cyan]"); result = asyncio.run(engine.verify(reproduction_result)). except ValueError as e → console.print error, raise typer.Exit(code=1)
6. **Display with Rich (VerificationResult-specific):**
   - status_color = "green" if result.status == "pass" else ("yellow" if result.status == "no_tests" else "red")
   - console.print(f"\n[bold]Verification Result[/bold]")
   - console.print(f"Status: [{status_color}]{result.status}[/{status_color}]")
   - console.print(f"Framework: [cyan]{result.framework or 'none detected'}[/cyan]")
   - console.print(f"Command: [dim]{result.command}[/dim]")
   - console.print(f"Tests found: {result.tests_found} | Tests run: {result.tests_run}")
   - console.print(f"Passed: [green]{result.passed}[/green] | Failed: [red]{result.failed}[/red] | Errors: [red]{result.errors}[/red]")
   - If result.skipped > 0: console.print(f"Skipped: [yellow]{result.skipped}[/yellow]")
   - console.print(f"Duration: {result.duration_seconds:.2f}s")
   - console.print(f"\n[bold]Summary:[/bold] {result.summary}")
   - If result.files_tested: console.print(f"\n[dim]Test files executed:[/dim]"); for tf in result.files_tested: console.print(f"  • {tf}")
   - If result.tests: display per-test details in a Rich Table with columns: Test, Outcome, Duration, Message (only for failed/error tests to avoid noise)
7. **Output JSON:** if output is not None: with open(output, "w") as f: f.write(result.model_dump_json(indent=2)); console.print(f"[green]Saved verification result to {output}[/green]")
8. **Error exit:** if result.status in ("fail", "error", "timeout"): raise typer.Exit(code=1) — exit non-zero so shell pipes / CI detect failure
