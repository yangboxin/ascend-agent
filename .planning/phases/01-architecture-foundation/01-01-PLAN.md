```python
from pydantic import BaseModel, ConfigDict, Field

class ConfigEnv(BaseModel):
    model_config = ConfigDict(extra="forbid")
    python_version: str = Field(default="")
    platform: str = Field(default="")
    env_vars: dict[str, str] = Field(default_factory=dict)

class RepoInfo(BaseModel):
    model_config = ConfigDict(extra="forbid")
    path: str
    language: str = "python"
    file_count: int = 0
    structure: list[str] = Field(default_factory=list)

class TraceEntry(BaseModel):
    model_config = ConfigDict(extra="forbid")
    file: str | None = None
    line: int | None = Field(None, ge=1)
    function: str | None = None
    text: str

class TraceInfo(BaseModel):
    model_config = ConfigDict(extra="forbid")
    error_type: str | None = None
    error_message: str | None = None
    frames: list[TraceEntry] = Field(default_factory=list)
    raw_text: str

class ContextDocument(BaseModel):
    model_config = ConfigDict(extra="forbid")
    repo: RepoInfo | None = None
    trace: TraceInfo | None = None
    config_env: ConfigEnv = Field(default_factory=ConfigEnv)
```

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Tampering | context/models.py | mitigate | All models use `ConfigDict(extra="forbid")` — rejects unknown fields. Field types enforce constraints (`ge=1` on line numbers). String fields prevent injection at the schema contract level. |
| T-01-02 | Tampering | pip installs | mitigate | All packages from RESEARCH.md Package Legitimacy Audit are [Approved] — established projects (Anthropic, tiangolo, Textualize, pydantic). No [SUS] or [SLOP] packages. No human-verify gate needed per audit. |
| T-01-03 | Information Disclosure | config.py (Settings) | accept | Settings captures `os.environ` dict which may contain secrets. This is an intentional design choice per D-12 (config env schema). The ContextDocument is consumed locally by the diagnosis engine (Phase 2) and not transmitted externally. Accept this risk. |
